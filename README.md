OPC UA Client Logger

This project implements an OPC UA server and client using Python.
The client connects to the server, reads 10 simulated OPC UA tags
every 60 seconds, and logs the data into hourly CSV files.

Features:
- Python-based OPC UA server simulation
- OPC UA client with address space browsing
- Periodic data logging (60 seconds)
- Hourly CSV file rotation
- Timestamp and epoch logging
